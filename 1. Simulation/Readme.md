This folder contains the code to reproduce the results of chapter 5.1. 

It contains two folder:
- The folder "Step 1: run algorithms" creates instances and runs the MPL-BD algorithm and the MJ-MCMC algorithm on these instances. It produces .Rdata files.
- The folder "Step 2: read results" reads the .Rdata files created in "step 1" and produces the figures in chapter 5.1 and in the supplemental material.
