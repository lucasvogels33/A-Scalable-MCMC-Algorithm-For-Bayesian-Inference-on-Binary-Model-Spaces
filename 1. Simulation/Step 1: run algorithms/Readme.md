This folder contains the following files:

1. MPLRJ_jump_functions - contains the code for the MJ-MCMC algorithm
2. MPLBD_funcions - contains the code for the MPL-BD algorithm
3. metric_functions - contains the code to calculate the AUC-PR, AUC-ROC, p+ and p-
4. Run_and_save - this is the main Rscript. It uses all other Rscripts. It creates an instance (data, true graph), runs an algorithm, and saves the results in an .Rdata file
