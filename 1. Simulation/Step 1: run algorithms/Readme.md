This folder contains the following files:
1. Run_and_save - this is the main Rscript. It uses all other Rscripts. It creates an instance (data, true graph), runs an algorithm, and saves the results in an .Rdata file
2. 1. MPLRJ_jump_functions - contains the code for the MJ-MCMC algorithm
3. 2. MPLBD_funcions - contains the code for the MPL-BD algorithm
4. 3. metric_functions - contains the code to calculate the AUC-PR, AUC-ROC, p+ and p-
