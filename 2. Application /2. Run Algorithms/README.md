This folder contains the following files:

1. MPLRJ_jump_functions - contains the code for the MJ-MCMC algorithm
2. MPLBD_funcions - contains the code for the MPL-BD algorithm
Run_and_save - this is the main Rscript. It uses all other Rscripts. It loads the cleaned and prepared data, runs an algorithm, and saves the results in an .Rdata file
