This folder creates the code to recreate the figures in Section 5.2 (the application to mice immune genes)

It contains three folders:
1. Prepare data - this folder uses online data, cleans them and prepares them
2. Run algorithms - this folder uses the cleaned data from the previous step and runs the MJ-MCMC and BD-MPL algorithms. It creates .Rdata files with the results
3. Read results - this folder reads the .Rdata created in the previous step and creates the figures in Section 5.2
