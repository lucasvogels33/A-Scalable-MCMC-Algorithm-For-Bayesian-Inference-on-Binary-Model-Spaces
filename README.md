# A-Scalable-MCMC-Algorithm-For-Bayesian-Inference-on-Binary-Model-Spaces
This repository contains all files required to reproduce the results in the article "A Scalable MCMC Algorithm For Bayesian Inference on Binary Model Spaces" by Lucas Vogels et al. It contains two folders:

The folder "1.Simulation" contains the code to reproduce the results of the simulation study (i.e. chapter 5.1).
The folder "2.Application" contains the code to reproduce the results of the simulation study (i.e. chapter 5.2).
